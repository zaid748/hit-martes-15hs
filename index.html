Areas de Git

Working Directory

El area de trabajo hace referencia a la carpeta donde creamos el repositorio. muy rara vez utilizamos la linea de comandos en esta zona, pero es donde realizaremos todas las modificaciones relacionadas con nuestro proyecto

Staging Area (INDEX)

El area de control de cambios corresponde al archivo que se encuentran en la carpeta .git que se crea al iniciar un repositorio. Aqui se regitran los archivos a los que le vamos a hacer seguimiento y es el punto de comparacion para confirmar dichos cambios. Es necesario agregar los archivos con los que estemos trabajando a dicha area para avanzar con el historial de cambios.
